# UCLA at Baruch College - Official Website

Welcome to the official website repository for **UCLA at Baruch College**, a student organization dedicated to promoting Chinese culture and fostering a supportive community. We organize fun events to educate and engage people with Chinese traditions and culture.

## Project Overview

This website is built to showcase UCLA at Baruch's activities, events, and mission. The website is a key tool for informing students and visitors about our work, upcoming events, and providing insight into Chinese culture.

### Website Sections

- **Navigation Bar**: Interactive navigation bar to access sections faster.
- **Pillars**: Showcase the core pillars UCLA club follows.
- **About Us**: Provides an overview of our club, its mission, and upcoming events.
- **Events**: Displays information about our cultural events and initiatives.
- **Culture**: Educates visitors on Chinese traditions, festivals, and cultural highlights.
- **Board Members**: Introduces the team behind UCLA at Baruch.
- **Footer**: Contains contact information and links to our social media.

## Technologies Used

- **Frontend Framework**: React (JavaScript library)
- **Styling**: Custom CSS
- **Development Tool**: Vite for fast build times and hot module reloading

## Getting Started

To run the project locally, follow the steps below:

### Prerequisites

Make sure you have **Node.js** and **npm** installed on your machine.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/baruch-ucla-website.git

2. Navigate to the project folder:
   ```bash
   cd ucla-website
3. Install dependencies:
   ```bash
   npm install
4. Run the development server:
   ```bash
   npm run dev
